# Product Brief: Flotera
**Nume Proiect:** Flotera
**Proprietar:** Alex
**Status:** Finalizat

## Viziune
O platformă profesională și rafinată de gestionare a flotelor auto pentru micii antreprenori, destinată urmăririi mentenanței vehiculelor, expirării documentelor legale (ITP, RCA) și a kilometrajului, printr-o interfață extrem de vizuală și intuitivă.

## Public Țintă
Proprietari de mici afaceri (3-15 vehicule) care doresc să evite amenzile și să mențină starea tehnică a vehiculelor fără complexitatea instrumentelor de tip enterprise.

## Caracteristici Cheie
1. **Dashboard Vizual al Vehiculelor:** Interfață bazată pe carduri cu fotografii reale ale vehiculelor pentru identificare rapidă.
2. **Clopoțel de Notificări:** Un centru de notificări dedicat în colțul din dreapta-sus.
3. **Alerte Persistente:** Notificări pentru ITP, RCA și mentenanță care pot fi marcate ca "citite".
4. **Status tip Semafor:** Indicatori vizuali (Verde/Galben/Roșu) în funcție de proximitatea datelor de expirare.
5. **Urmărire Kilometraj:** Jurnal simplu pentru actualizarea kilometrajului.

## Stivă Tehnologică
- **Backend:** Java (Spring Boot)
- **Frontend:** React
- **Bază de date:** PostgreSQL (cu cale de migrare viitoare către Supabase)
- **Deployment:** Țintă viitoare - Vercel

## Metricii de Succes (Proiect Universitar)
- Interfață curată, "drăguță" dar profesională.
- Persistența notificărilor complet funcțională.
- Backend Java robust cu contracte API clare.
